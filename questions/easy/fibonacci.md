# 构建一个数组，内容到第 n 项的 Fibonacci 序列

## 参考答案

```es6
const fibonacci = n =>
  [...Array(n)].reduce(
    (acc, val, i) => acc.concat(i > 1 ? acc[i - 1] + acc[i - 2] : i),
    []
  )
```
