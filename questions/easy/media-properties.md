# 你能说出四类 `@media` 的属性么

## 参考答案

* `all`, 适用于所有媒体类型设备
* `print`, 仅适用于打印机
* `screen`, 仅适用于屏幕（台式机，平板电脑，手机等）
* `speech`, 仅适用于屏幕阅读器
