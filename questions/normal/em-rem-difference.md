# 在 CSS 中 `em` 与 `rem` 有什么区别

## 参考答案

`em` 和 `rem` 都基于 `font-size` CSS属性，唯一的区别是他们从哪里继承的值。

* `em` 单位从父元素的`font-size`继承它们的值；
* `rem`单元从根元素的`font-size`继承它们的值（`html`）。

在大多数游览器中，根元素的 `font-size` 默认值为 `16px`。
